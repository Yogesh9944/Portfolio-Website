# 🚀 Yogesh Pande — Developer Portfolio

A clean, responsive personal portfolio website built with **React + Vite**, designed to showcase projects, skills, and experience to recruiters and hiring managers.

🌐 **Live Site:** ["https://portfolio-website-jhm5.vercel.app/"]

---

## ✨ Features

- ⚡ Fast performance with Vite bundling
- 🎨 Clean dark UI with smooth scroll animations
- ⌨️ Typewriter effect in Hero section
- 📱 Fully responsive — mobile, tablet & desktop
- 🔍 SEO-optimised with meta tags
- 📬 Contact form with email integration
- 🧩 Scroll-reveal animations on all sections

---

## 🚀 Getting Started

### Prerequisites
- Node.js v18+
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/yogeshpande/yogesh-portfolio.git

# Navigate into the project
cd yogesh-portfolio

# Install dependencies
npm install

# Start development server
npm run dev
```

Open [https://portfolio-website-jhm5.vercel.app/] in your browser.

### Build for Production

```bash
npm run build
```

## 🔧 Customisation

To personalise this portfolio for yourself, update the following:

| File | What to update |
|---|---|
| `src/components/Hero.jsx` | Name, resume link, social URLs |
| `src/components/About.jsx` | Bio, stats, education details |
| `src/components/Skills.jsx` | Skill names and proficiency levels |
| `src/components/Projects.jsx` | Project titles, descriptions, live & GitHub links |
| `src/components/Experience.jsx` | Work experience, certifications |
| `src/components/Contact.jsx` | Email, LinkedIn, GitHub links |
| `index.html` | Meta title and description |

---

## 📦 Deployment (Vercel)

1. Push your code to a GitHub repository
2. Go to [vercel.com](https://vercel.com) and sign in
3. Click **Add New Project** → import your repo
4. Leave all settings as default
5. Click **Deploy** — done in under a minute ✅
